Installation

Follow the steps below to get started with your Site Template:
Open the Package/HTML Folder to find all the Templates Files You will need to Upload these files to your Web Server using FTP in order to use it on your Website. Make sure you upload the required files/folders listed below:

  1. HTML/css - Extra Stylesheets Folder
  2. HTML/img - img Folder
  3. HTML/js - Javacripts Folder
  4. HTML/index.html - Index File/Homepage
The other files can be used according to your preferences. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Getting CSS

Css files

There are 2 main CSS files in this template.

  • style.css - template common styles
  • responsive.css - Responsive design for all device
  • slick.css - For slider
  • bootstrap-v4.3.0.min.css
  • flaticon.css

Note:

  • Make sure the responsive.css is below other css files

Getting JavaScript

This template imports many Javascript files. All located in the "js" folder.

  1. jquery.1.12.4.js
  2. bootstrap.min.js
  3. slick.min.js
  4. custom.js

Note:

  1. Make the custom-script.js at last (below other js files).

Color Schemes

You can change your Website's Color Scheme in an instant. You simply need to change the HEX Color Code in the css/home1.css , css/home2.css ,css/home3.css file & you are good to go. Follow these quick steps to get going:

	...

	:root {
		--main-color: #402bb2;
	}
	...

Now simply change the HEX Codes according to your requirements.
